I/Introduction

1-My SQL

  • My SQL peut s'utiliser en ligne de commande (Shell,Bash,CMDR et le terminal sur Mac ou Linux,ou avec une autre interface graphique comme phpMyAdmin
  • Pour se connecter en ligne de commande on utilise my sql - u-utilisateur-p
  • Attention:Il faut penser à mettre le nom d'utilisateur spécifique au serveur ; sur le PC en local l'utilisateur est : root qui signifie : 'racine'
  • En local,sur PC il n'y a pas de mot de passe,sur Mac le mdp est 'root'
  • En ligne chaque BBD aura son utilisateur propre et son mot de passe propre

2-SGBDR ou SGBD

  • Un SGBDR (SGBD)
  • Un systéme qui permet de gérer une BDDQ Bas de données
  • CRUD:create , read ,update,delete
  • Date de naissance 1994
  • Open Source
  • Concurrents : oracle , Ms Access ,SQLite,PostgreSQL
  • SGBDR : systéme de gestion de base de données relationnelles
  • SGBD est systéme de gestion de base de données
  • MCD Modéle conceptuel de données"
  • BDD:Bas de Données
  • DB est Date Bas
  • SQL structured Query Language

3-En résumé

  • a l'intérieur de la BDD les informations sont classées, structurées et regroupées par sujet
  • le langue de requête SQL nous permet d'échénger les informations avec la base; autrement dit il est utilisé pour la communication entre le client et le serveur
  • dans une BDD les données sont représentées sous formr de "table"
  • dans une table nous avons des colonnes, chaques données sera introduite sous forme de ligne dans la table
  • une fois que les informations ont été enregistrées, il est important de les gérer (ajout, modification, suppression, consultation)
