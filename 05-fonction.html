Base de données pour PHP Bibliothèques

1- Quelques fonctions

Tableau Fonctions prédéfinies
Fonctions prédéfinies Description
SELECT CURDATE() Affiche la date courante : quand on aura besoin de la date du jour dans la requête
SELECT CURDATE() +0 Affiche la date courante sans tirets, pratique pour nommer un fichier
SELECT CURTIME() Affiche l'heure
SELECT NOW() Affiche la date et l'heure
SELECT DATE_ADD('2021-12-06', INTERVAL 31 DAY) Affiche xx jour à la date de départ
SELECT date_rendu, DATE_FORMAT(date_rendu, '%d/%m/%Y') AS 'date FR' FROM emprunt Affiche la date en format francais
SELECT CONCAT_WS('-', 'a', 'b', 'c') WS = With separator utile pour concatener un code postal avec une ville
SELECT TRIM(' coucou') Supprime les espaces en debut et en fin de chaine
